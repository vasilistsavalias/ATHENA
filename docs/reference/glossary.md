# Glossary

- **ATHENA** — the overall project and repository
- **FT-SD** — fine-tuned Stable Diffusion inpainting model
- **TTA** — test-time augmentation
- **BLIP2** — captioning model used in `S07`
- **Qwen** — language model used for caption refinement
- **LaMa / MAT / CoModGAN** — deep inpainting baselines
- **Prompt family** — `Unconditional`, `Raw Text`, `Enriched Text`, or `Refined Text`
- **Prompt-ready sample** — a sample with usable raw, enriched, and refined captions
- **Run-state ledger** — `outputs/00_logs/run_state.json`
- **Strict mode** — fail-fast behavior driven by config and prerequisite gates
- **Smoke test** — reduced-scale real-data run
- **Expert pack** — curated image set imported into the website study
